# iOS

## Set up manually

1. Open Settings
2. Click on Wi-Fi
3. Pick a Wi-Fi network and click on the blue i icon
4. Click Configure DNS and select Manual
5. Click Add Server and add these DNS addresses:

### IPv4

```
185.231.233.158
```

### IPv6 (optional)

```
2001:470:1f20:eb::2
```
